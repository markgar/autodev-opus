# Milestone 07a — Sample Specs Blob Storage Service and CRUD API

## Milestone: Sample specs blob storage service and CRUD API routes

> **Validates:** `npm run build` succeeds. `npm start` starts the server. `GET /api/sample-specs` returns HTTP 200 with a JSON array (empty if no specs exist). `POST /api/sample-specs` with JSON body `{ "name": "test.md", "content": "# Hello" }` returns HTTP 201. `GET /api/sample-specs/test.md` returns HTTP 200 with text content `# Hello`. `DELETE /api/sample-specs/test.md` returns HTTP 204. After delete, `GET /api/sample-specs` returns an empty array. `npm test` passes all existing tests.

> **Reference files:** `src/server/routes/health.ts` (existing route pattern — Router export, mounted under /api), `src/server/azure/blobClient.ts` (shared BlobServiceClient instance to use for blob operations), `src/server/app.ts` (where to register new router with app.use), `src/client/index.css` (theme CSS variables to extend).
